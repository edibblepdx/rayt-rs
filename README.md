# rayt-rs

**A multithreaded software ray tracer written in Rust.**

## Features
- [x] Sphere intersection
- [ ] Gltf model loading
- [ ] Materials
- [x] Camera
- [x] Multithreaded rendering using rayon
- [ ] Shadows
- [ ] Scene loading from toml file
- [x] Deserialization of geometric primitives
